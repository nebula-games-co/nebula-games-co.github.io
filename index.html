Instrucciones

  1. Genera un código de conexión haciendo clic en el botón "Generar Nuevo Código"
  2. Abre tu juego de ritmo en Roblox e introduce el código cuando se solicite
  3. Una vez conectado, tu juego podrá enviar peticiones HTTP a esta página para reproducir audio
  4. Las canciones disponibles aparecerán en la lista inferior cuando estés conectado

Nota: Esta conexión funciona solo localmente y es necesaria para cada sesión de juego.
